# NeuralMesh 3D Planner

O **NeuralMesh 3D Planner** é uma ferramenta de planejamento interativa em 3D projetada para visualizar arquiteturas de microsserviços como uma rede neural. Esta aplicação permite modelar conexões complexas, simular fluxos de rede e otimizar topologias de sistemas com uma experiência visual imersiva de alta performance.

---

## 🚀 Instalação e Execução Rápida

Para facilitar a execução local do projeto, disponibilizamos um script de automação (`setup.sh`) que realiza a instalação de todas as dependências e compila o sistema de forma automatizada.

### Como Executar:

1. **Permissão de Execução** (caso esteja em sistemas Unix/Linux/macOS):
   ```bash
   chmod +x setup.sh
   ```

2. **Iniciar Instalação e Compilação**:
   ```bash
   ./setup.sh
   ```
   *Este script executará de forma automatizada o `npm install` seguido pelo `npm run build`.*

3. **Executar o Servidor de Produção**:
   ```bash
   npm start
   ```

---

## 🛠️ Execução Manual

Caso prefira executar as etapas manualmente, siga os comandos abaixo:

1. **Instalar Dependências**:
   ```bash
   npm install
   ```

2. **Compilar Aplicação (Frontend e Backend)**:
   ```bash
   npm run build
   ```

3. **Iniciar Servidor**:
   ```bash
   npm start
   ```
